# 🗃️ Employee Management System — MySQL

## 📌 Objective
Design and develop a normalized relational database to efficiently manage employee, department, salary, and leave data.

## 🧱 Database Structure
| Table | Description |
|---|---|
| JobDepartment | Job roles, departments, salary ranges |
| SalaryBonus | Salary, bonus, annual pay per job role |
| Employee | Personal, contact details of employees |
| Qualification | Employee qualifications and skills |
| Leaves | Employee leave records |
| Payroll | Net payment calculations |

## 🔍 Key Insights
- Finance dept has highest bonus allocation: $96,000
- Legal dept offers highest average salary: $84,600
- Total monthly payroll: $2.78M across 60 employees
- 100% employees have qualifications documented
